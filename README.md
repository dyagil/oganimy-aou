# Pipedrive-JotForm Bridge

שירות המקשר בין Pipedrive ל-JotForm. השירות מאזין לעדכונים מ-Pipedrive, מושך פרטי איש קשר, ויוצר קישור מותאם אישית לטופס JotForm עם הפרטים המוזנים מראש.

## תכונות

- מאזין לעדכונים מ-Pipedrive דרך webhook
- מושך פרטי איש קשר מ-Pipedrive API
- יוצר קישור מותאם אישית לטופס JotForm
- יוצר משימה חדשה ב-Pipedrive עם הקישור לטופס

## דרישות

- מפתח API של Pipedrive (מוגדר כמשתנה סביבה `PIPEDRIVE_API_KEY`)

## פריסה

השירות מוכן לפריסה ב-Railway.app.
